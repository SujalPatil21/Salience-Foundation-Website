# Salience Foundation — Cinematic Website

A modern, cinematic storytelling website built for Salience Foundation, a non-profit focused on empowering rural talent in India through sports, arts, and cultural development.

---

## Description

This project transforms a traditional NGO website into a high-impact digital experience using storytelling, smooth animations, and real-world visuals.

The website is designed to:
- Highlight the challenges faced by rural talent  
- Showcase potential and real impact  
- Drive user action (athlete registration, volunteering, support)
